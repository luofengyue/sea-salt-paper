# 海盐折纸小程序

这是一个基于微信小程序的《海盐折纸》轻量单机练习版原型。项目目前以本地练习和静态页面为主，包含首页大厅、规则教程、牌库图鉴、房间等待页、AI 练习页和结算页。

> 说明：这是非官方学习项目，不隶属于原桌游发行方。

## 功能概览

- 首页大厅入口
- 规则教程页面
- 测试牌库图鉴
- 静态好友房等待页
- 单机 AI 练习原型
- 回合结算页面

## 本地运行

1. 安装微信开发者工具。
2. 克隆或下载本仓库。
3. 打开微信开发者工具，选择“导入项目”。
4. 项目目录选择：

```text
miniprogram
```

5. AppID 可以使用测试号，也可以替换成你自己的小程序 AppID。
6. 点击“编译”即可运行。

## 项目结构

```text
.
├── Doc/                 # 开发说明文档

│   ├── app.js
│   ├── app.json
│   └── project.config.json
├── package.json         # 可选的本地检查依赖
└── README.md
```

## 开发检查

如果你安装了 Node.js，可以在仓库根目录运行：

```bash
npm install
npm run check
```

这个检查只做基础 JavaScript 语法校验。小程序页面请以微信开发者工具编译结果为准。
